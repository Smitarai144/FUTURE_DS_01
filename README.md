# FUTURE_DS_01 – Sales Performance Analytics

## 📌 Project Overview
This project was completed as part of the Future Interns Data Science & Analytics Internship (Task 1). The objective is to analyze business sales data to identify revenue trends, top-selling products, high-value categories, and regional performance using Excel and Power BI.

## 🎯 Objectives
- Clean and prepare the sales dataset.
- Analyze sales performance using KPIs.
- Identify top-selling products and profitable categories.
- Compare sales across different regions.
- Create an interactive dashboard with business insights.

## 🛠️ Tools Used
- Microsoft Excel (Data Cleaning)
- Microsoft Power BI (Dashboard & Data Visualization)

## 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Sales by Category
- Top-Selling Products
- Regional Sales Performance
- Interactive Filters (Slicers)

## 🔍 Key Insights
- Identified revenue trends over time.
- Compared sales performance across regions.
- Analyzed top-selling products and high-value categories.
- Evaluated overall business performance using KPIs.

## 💡 Recommendations
- Focus marketing efforts on high-performing regions.
- Increase inventory for top-selling products.
- Improve strategies for low-performing categories.
- Monitor monthly sales trends for better decision-making.
